Task-4: Classification Models, Evaluation Metrics & Handling Imbalanced Data

📌 Overview
This project demonstrates the implementation of Machine Learning classification models using the Breast Cancer Wisconsin Dataset.

The project focuses on:

 Logistic Regression
 Decision Tree Classification
 Model Evaluation Metrics
 ROC Curve & AUC Score
 Handling Imbalanced Data

🗂️ Dataset Used

Breast Cancer Wisconsin Dataset from scikit-learn.

The dataset contains medical diagnostic features used for breast cancer classification.

🎯 Target Classes

0 = Malignant
1 = Benign

🛠️ Technologies Used

 Python
 Pandas
 NumPy
 Matplotlib
 Scikit-learn
 Jupyter Notebook

🤖 Models Implemented

1. Logistic Regression
Used as the primary classification model.

2. Decision Tree Classifier
Used for model comparison.

📊 Evaluation Metrics Used

The following evaluation metrics were implemented:

 Accuracy Score
 Confusion Matrix
 Precision
 Recall
 F1-score
 ROC Curve
 AUC Score

📉 Graphs Included

 Confusion Matrix Heatmap
 ROC Curve

📚 Key Learnings

 Difference between Regression and Classification
 Logistic Regression implementation
 Understanding Confusion Matrix
 Importance of Precision, Recall and F1-score
 ROC Curve and AUC interpretation
 Handling Imbalanced Data
 Comparing Multiple Classification Models

🏁 Conclusion

Logistic Regression achieved high accuracy and strong ROC-AUC performance on the Breast Cancer dataset.
Decision Tree Classifier also performed well but showed comparatively lower performance.
The project successfully demonstrates classification model building and evaluation techniques.
